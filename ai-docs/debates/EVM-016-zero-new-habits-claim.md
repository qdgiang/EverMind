# EVM-016 — Correct the “zero new habits” claim

> Priority: P2 · Status: `OPEN`

## Problem

Members avoid new weekly reporting forms, but leads still review proposals, exceptions, and
publication bundles. “Zero new habits” overpromises and hides a deliberate governance loop.

## Options

- **Option A — Keep the current phrase:** memorable but inaccurate.
- **Option B — Narrow the claim (`PROPOSED`):** “No new reporting destination for members; a lightweight
  exception-review loop for leads.”
- **Option C — Remove the claim:** safest but loses a useful explanation of ambient capture.

## Acceptance criteria

- Vision, pitch, demo script, and product copy use the same qualified claim.
- Copy distinguishes member reporting from lead governance.
- No flow claims to be habit-free when it requires explicit review.
